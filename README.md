## 本地使用

```bash
git clone https://github.com/OOOHGOD/Study_Notes.git
cd Study_Notes
```

克隆完成后，可以直接阅读 Markdown 文件，或在 Obsidian 中选择“打开本地仓库”，并指向该目录。部分笔记仍处于整理阶段，目录结构和内容会随着学习进度持续调整。
